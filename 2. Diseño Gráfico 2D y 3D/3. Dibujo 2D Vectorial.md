- El dibujo 2D Vectorial, se dibuja por vectores con los pixeles en la pantalla.
- Un ejemplo de como se define de una página de dibujo vectorial:
	<svg width="70mm" height="99mm" viewBox="0 0 70 99" xmlns="http://www.w3.org/2000/svg">
  <!-- Círculo centrado (antes 105,100,r=40 → ahora /3) -->
  <circle cx="35" cy="33.33" r="13.33" fill="none" stroke="white" stroke-width="1.5"/>
  <!-- Triángulo (todas las coordenadas divididas entre 3) -->
  <polygon 
    points="35,60 21.67,83.33 48.33,83.33"
    fill="none" stroke="white" stroke-width="1.5"/>
</svg>

- Extensiones de programas vectoriales
	- .svg para vectorial.
	- .ai  de adobe ilustrator.
	- .ps para PostScript.
		- Lenguaje de descripción de páginas, nació con las impresoras laser.
	- pdf de Portable File Document.
		- Mientras haya un visor, es compatible con cualquier sistema.
	- .dwg de AutoCad.
	- .gdi: Igual que .ps pero en windows.
-  El sistema operativo es quien gestiona las fuentes
